ABSTRACT

Necessity is the mother of all invention.This project uses a biometric concept to simplify
the attendance system in educational institutes.
It uses the most reliable way of distinctively
identifying students through fingerprint reading.
Such type of application is very useful in school
as well as in college for daily attendance.
Through this application we can keep a
systematic track of studentâ€™s attendance.
This project enables the easy way of
maintaining class attendance with fewer efforts.



Description

1. The project requires a fingerprint reader for finger detection.
2. Every student can login to the system through finger detection.
3. The fingerprint of the student is compared with the one stored
in database and if it matches then attendance is marked for that particular student.
4. The system also generates a brief report of attendance from the
database according to subject-wise or date-wise as required.
5. A defaulter list can be generated through system.
6. Admin has the option to take a print of the reports and defaulter list thus generated.



Advantages

1. Students will be more regular in attending their classes since
now no password or no attendance sheet signature is required,
so no friend or any other student can make an attendance on
behalf of others as fingerprints are unique for every student.
2. Teachers do not need to waste their time approximately 15min of
1hour for taking attendance of students.
3. No need to maintain attendance sheet as the attendance are electronically stored in.
4. The system helps the faculty to easily find out defaulters.
5. User may easily get attendance history of a particular student.
6. It saves time, cost, efforts and institute resources.



Security Features

1. Prepared-Statement to avoid SQL injections.
2. Error page to prevent the server information.
3. SHA-256 algorithm used for encryption of sensitive data like passwords.
4. Use of Re-captcha in forms to avoid attacks like cross site request forgery attack.
5. Use of one-time-password (OTP) to authenticate the user.
6. Prevention of spamming OTP using time based OPT mailer.
7. Session Management to prevent unauthorized access to data tables.
8. Menu can be only accessed by teachers/admins with their fingerprint.
9. Teacher fingerprint is needed to start lab, prevents illegal attendance marking by students.
10. Buzzer to alert teacher of unauthorized access, if any, detected on the system.
